Retrieve all entries in a conversation.


GET
 /v1/conversations/{conversation_id}/history

Given a conversation_id retrieve all the entries belonging to that conversation. The entries are sorted in the order they were appended, those can be messages, connectors or function_call.


Parameters
application/json

conversation_id
*
string

ID of the conversation from which we are fetching entries.

200

Successful Response

conversation_id
*
string


entries
*
array<MessageInputEntry|MessageOutputEntry|FunctionResultEntry|FunctionCallEntry|ToolExecutionEntry|AgentHandoffEntry>

object
"conversation.history"

Default Value: "conversation.history"


TypeScript


Python


cURL


curl https://api.mistral.ai/v1/conversations/{conversation_id}/history \
 -X GET \
 -H 'Authorization: Bearer YOUR_APIKEY_HERE'
200


{
  "conversation_id": "ipsum eiusmod",
  "entries": [
    {
      "content": "consequat do",
      "role": "assistant"
    }
  ]
}

Restart a conversation starting from a given entry.


POST
 /v1/conversations/{conversation_id}/restart

Given a conversation_id and an id, recreate a conversation from this point and run completion. A new conversation is returned with the new entries returned.


Parameters
application/json

conversation_id
*
string

ID of the original conversation which is being restarted.


Request Body
application/json

Request to restart a new conversation from a given entry in the conversation.

agent_version
string|integer|null

Specific version of the agent to use when restarting. If not provided, uses the current version.


completion_args
CompletionArgs

White-listed arguments from the completion API

from_entry_id
*
string

handoff_execution
"client"|"server"

Default Value: "server"


inputs
*
string|array<MessageInputEntry|MessageOutputEntry|FunctionResultEntry|FunctionCallEntry|ToolExecutionEntry|AgentHandoffEntry>

metadata
map<any>|null

Custom metadata for the conversation.

store
boolean

Default Value: true

Whether to store the results into our servers or not.

stream
boolean

Default Value: false

200

Successful Response

conversation_id
*
string

object
"conversation.response"

Default Value: "conversation.response"


outputs
*
array<MessageOutputEntry|ToolExecutionEntry|FunctionCallEntry|AgentHandoffEntry>


usage
*
ConversationUsageInfo


TypeScript


Python


cURL


curl https://api.mistral.ai/v1/conversations/{conversation_id}/restart \
 -X POST \
 -H 'Authorization: Bearer YOUR_APIKEY_HERE' \
 -H 'Content-Type: application/json' \
 -d '{
  "from_entry_id": "ipsum eiusmod",
  "inputs": "consequat do"
}'
200


{
  "conversation_id": "ipsum eiusmod",
  "outputs": [
    {
      "content": "consequat do"
    }
