# Components Overview

## Common Components

### ErrorBoundary

A React error boundary component that catches uncaught errors and displays a user-friendly error page with error details and recovery options.

### Logger

A lightweight, extensible logging library with support for multiple log levels, scoping, and custom transports.

### LanguageSwitcher

A dropdown component for switching between languages, with visual indicators for static vs backend-translated languages and loading states.

### TranslationExample

A demo component showcasing translation functionality across different namespaces (common, navigation, game) with loading indicators.
